## 🔰 About
Scoolors is a personal project focused on generating modern, perceptually balanced color systems starting from a single base color.

The application creates:

* Primary color scales
* Neutral tinted scales
* Dynamic CSS variables
* Exportable design tokens in multiple formats

The goal is to provide developers and designers with a practical tool for generating consistent UI-ready color palettes.

Direct link to the project: <a href="https://sp1lu.github.io/scoolors/" target="_blank">https://sp1lu.github.io/scoolors/</a>

## ✨ Features

Current main features include:

- [x] Dynamic OKLCH color scale generation
- [x] Neutral UI palette generation
- [x] Real-time CSS variable generation
- [x] Export in multiple color spaces
- [x] Responsive UI
- [x] Syntax-highlighted code export
